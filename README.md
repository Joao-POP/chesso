# Sobre

Chesso é um cliente de xadrez virtual.

# Funcionalidades

1. Suporte a arquivos PGN.
2. Partidas de turno local.
3. Suporte a mouse.
4. Partida online P2P.
5. Botão de checkmate instantânce.
